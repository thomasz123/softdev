GET VS POST

Get requests request data while post requests sends data. Get displays the username (input) in the url like this: /auth?username=a&sub1=Submit. Post doesn't have the username in the url, it is just /auth. It also has a resubmit popup everytime you refresh the page. Get requests are stored in the browser's history while post requests are not.
